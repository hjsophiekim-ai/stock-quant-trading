정확히 말하면, 드롭다운 기준으로는 이렇게 보시면 됩니다.

시간대로 풀어드리면:

**1) `swing_relaxed_v2` — 메인 스윙**
이건 장중 몇 시에 사고파는 전략이라기보다 **일봉 기반 스윙**입니다.
즉,

* 당일 분봉 타이밍 매매용이 아니라
* 일봉 조건 충족 시 진입/청산 판단
* 여러 날 보유 가능
* TP1/TP2/ATR 손절/MA20+ATR trailing/time exit 구조
  라서, “몇 시부터 몇 시까지”라고 보기보다 **하루 1틱 스윙 평가**에 가깝습니다. 스윙 전략으로 paper loop에 연결돼 있고, `swing_relaxed_v2`는 별도 TP1 상태 파일도 붙여서 돌립니다. 

**2) `final_betting_v1` — 메인 종가베팅**
이건 시간이 아주 명확합니다.

진입:

* 기본 **15:10 ~ 15:18**에만 신규 진입합니다. 설정 기본값이 그렇게 잡혀 있습니다.  

청산:

* 다음 거래일 오전 **09:00 ~ 10:30** 구간에서 청산 로직이 작동합니다. 기본 exit deadline은 10:30입니다.   
* 세부적으로는

  * **09:10 이후** ATR 기준 갭 하락이면 손절 가능
  * **09:12 ~ 10:00** 약한 오전 흐름이면 flush stop
  * **09:05 ~ 09:30** 갭 상승이면 차익실현 가능
  * **10:00 이후** 애매하면 value exit
  * **10:30 이후** hard exit
    입니다. 

즉 `final_betting_v1`은
**오늘 15:10~15:18에 사서, 내일 09:00~10:30에 정리하는 전략**이라고 보면 됩니다.
이건 장후반용입니다.

**3) `scalp_macd_rsi_3m_v1` — 메인 장중 3m MACD/RSI**
이건 장중 단타용입니다.

기본 시간:

* KRX 정규장 기준 **09:00 ~ 15:30** 안에서 움직입니다. 정규장 기본값이 09:00 오픈, 15:30 클로즈로 설정돼 있습니다. 
* 다만 진입은

  * 장 시작 후 **10분 동안 금지**
  * 장 마감 전 **20분 동안 금지**
    입니다. 기본 설정값이 그렇게 되어 있습니다. 
* 그래서 실제 신규 진입 가능 시간은 기본적으로
  **09:10 ~ 15:10 전후**
  로 이해하시면 됩니다.
* 또 장 마감 **15분 전 강제청산** 설정이 있어서, 실질적으로는 종가까지 끌고 가지 않고 당일 정리 성격이 강합니다.  

즉 이건
**장중 09:10부터, 너무 늦기 전까지 3분봉으로 치는 보수형 단타**입니다.

**4) `scalp_rsi_flag_hf_v1` — RSI red/blue 플래그 고빈도(3m)**
이게 제가 앞에서 말한 “하루 안에 여러 번 거래”에 가장 가까운 전략입니다.
설명도 코드에 직접
**RSI red/blue 플래그 기반 고빈도 스캘프 v1 (3m 봉 권장)**
이라고 되어 있습니다. 

성격:

* 3분봉
* 매수는 **RSI red flag**
* 매도는 **RSI blue flag 우선**
* 이후 손절/익절/트레일/시간청산/장마감 강제청산
* 종목별 일일 매수 상한 기본 **4회**
* 동시 보유 상한 기본 **4개**
  입니다.    

시간:

* 정규장 내에서 돌고
* 3분봉으로 실행되고
* 장마감 전 강제청산됩니다.  
* 이 전략 파일 자체에는 `scalp_macd`처럼 “장 시작 후 10분 금지 / 마감 전 20분 금지” 하드 블록은 안 보입니다.
  그래서 현재 코드 기준으로는
  **정규장 동안 비교적 넓게 도는 고빈도 3분봉 전략**으로 보는 게 맞습니다.
  다만 실제 주문은 session gate, flatten before close, risk halt, max trades/day 같은 공통 리스크 제약을 받습니다.  

**5) `intraday_rsi_flag_hf_v1` — 동일 로직(별칭 ID)**
이건 이름만 다르고 실질적으로는
**`scalp_rsi_flag_hf_v1`와 동일**합니다.
코드에서 둘 다 같은 전략 인스턴스를 씁니다. 

**6) `scalp_momentum_v2`, `scalp_momentum_v3`**
이건 지금 UI에도 실험/보조라고 적혀 있고, 메인으로 고르라고 만든 축은 아닙니다. 현재 구조상 실험용 보조입니다. 

그래서 지금 질문에 딱 답하면:

**하루 안에 여러 번 거래하고 싶다**
→ **`scalp_rsi_flag_hf_v1`** 고르세요.
정확히는

* **정규장 장중**
* **3분봉**
* **RSI red flag 매수 / blue flag 매도**
* **당일 여러 번 회전**
  용입니다. 

**오늘 장 막판에 사고 내일 아침 정리하고 싶다**
→ **`final_betting_v1`**

* 진입: **15:10~15:18**
* 청산: 다음날 **09:00~10:30**    

**하루 중 장중만 보수적으로 하고 싶다**
→ **`scalp_macd_rsi_3m_v1`**

* 진입 가능: 대략 **09:10 ~ 15:10 전후**
* 3분봉
* 장마감 전 정리   

**몇 시부터 몇 시까지인지 한 줄 정리**

* `swing_relaxed_v2`: 시간대 전략 아님, **일봉 스윙**
* `final_betting_v1`: **15:10~15:18 진입 / 다음날 09:00~10:30 청산**
* `scalp_macd_rsi_3m_v1`: **대략 09:10~15:10 장중 3분봉**
* `scalp_rsi_flag_hf_v1`: **정규장 장중 3분봉 고빈도**
* `intraday_rsi_flag_hf_v1`: **위와 동일한 별칭**

제가 지금 드롭다운에서 하나만 고르라고 하면,
**“장중 여러 번 거래” 목적이면 `scalp_rsi_flag_hf_v1`**
**“장후반 한 번 크게”면 `final_betting_v1`** 입니다.
